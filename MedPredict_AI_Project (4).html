Project Overview

Understanding the problem, goals, and dataset.

⚕️

Problem Statement

Manual medical diagnosis is time-consuming, expensive, and prone to human error. There is a growing need for AI-assisted decision support tools that can help clinicians and patients predict diseases from routine health parameters quickly and accurately.

🎯

Project Goal

Build a multi-class machine learning classification system that predicts a patient's primary diagnosis from their demographic data, vital signs, lab results, and symptoms — achieving over 95% accuracy.

📌

Scope

This project covers 10 disease categories including Asthma, Diabetes, Hypertension, Pneumonia, CAD, Migraine, Gastritis, Anemia, Viral Flu, and Healthy. It is intended as a clinical decision support tool, not a replacement for professional diagnosis.

Project Objectives

  1. Analyze and preprocess a real-world medical dataset with 1,000 patient records and 30 features.
  2. Perform comprehensive Exploratory Data Analysis (EDA) to identify key patterns and correlations.
  3. Train and compare four machine learning models: Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting.
  4. Evaluate models using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix.
  5. Select the best-performing model and build an end-to-end prediction pipeline.
  6. Develop a simple web interface for real-time disease prediction from patient inputs.

✅ Why Gradient Boosting Wins

Gradient Boosting sequentially builds 100 trees, each correcting the errors of the previous. This ensemble approach captures complex, non-linear interactions between features (e.g., HbA1c + BMI + age for diabetes detection) that simpler models miss. It achieves the highest accuracy (98%) and F1-Score (0.9798) with excellent per-class performance across all 10 disease categories.
